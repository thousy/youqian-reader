# YouQian书籍阅读器 📚

[![Version](https://img.shields.io/badge/version-1.4.3-blue.svg)](package.json)
[![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)](package.json)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](package.json)

**YouQian书籍阅读器** 是一款专为极致阅读体验打造的桌面电子书阅读器。它完美支持 **EPUB, MOBI, AZW3, PDF, TXT** 等主流电子书格式，配备了如纸质实体书般优雅的排版系统，拥有极快的书籍加载速度和顺畅的翻页交互，让您沉浸于纯净、舒适的阅读世界。

---

## 🌟 核心功能特色

### 1. 毫秒级极速“秒开”体验 ⚡
* **告别开书卡顿**：对大体积、多插图的 MOBI 和 AZW3 格式进行了底层深度性能调优，即使是数百万字、包含数百张插图的套装大书，也能实现**毫秒级“瞬开”**，体验与 EPUB 完全一致。
* **阅读翻页零延迟**：引入了高性能的节点缓存和延迟处理机制，让您翻页时页面瞬间完成响应，没有一丝粘滞感。对于超长篇幅的书籍，系统还会自动启用极速模式，确保流畅度始终处于最佳状态。

### 2. 像读纸质书般的优雅排版 📖
* **实体书横向分栏**：采用多栏流式排版，文字会自动分配在左右视口中，像展开的纸质书一样阅读，符合中国人的经典视线移动习惯。
* **三种排版模式一键切换**：阅读页顶栏右侧新增了直观的快捷切换按钮，您可以一键在以下三种模式间自由切换：
  * **全窗口**：经典优雅的单页/多栏版式。
  * **左右滚动**：带跨栏纸张质感的横向翻页体验。
  * **上下滚动**：连续的垂直瀑布流滚动体验。
* **个性化视觉定制**：通过阅读设置面板（“Aa”图标按钮），可以自由调整字号、字体、行高以及切换背景主题（明亮、护眼麦香、深邃暗黑等），完美适配各种阅读场景。

### 3. 智能好用的注释与旁注系统 💬
* **鼠标跟随气泡预览**：自动将书中复杂的注脚转化为优雅的纯文字上标。当鼠标悬浮在注号上时，会瞬间弹出内容小气泡，且气泡会根据鼠标的位置自动调整方向（偏左时显于右，偏右时显于左），智能防遮挡，鼠标离开即可自动消失。
* **章节注释一览面板**：在每个章节的正文结束、下一章节开始前，系统会为您自动整合一个具有毛玻璃质感的【本章注释】面板，集中展示该章节所有的注释。您不用再频繁翻到全书最后去查看注释，看完当前章节即可顺带一览，省心省力。

### 4. 100% 精准的目录与章节追踪 📍
* **绝不迷路**：针对 MOBI 和 AZW3 书籍的特殊结构进行了底层修正，完美解决了传统阅读器中“点击目录超链接无法跳转”或“底部章节名始终显示为正文”的问题。
* **实时章节同步**：无论您是通过目录跳转，还是手动滑页，底部的状态栏都会 100% 准确地同步显示您当前正在阅读的章节名称和页码进度。

### 5. 安全的数据备份与还原功能 💾
* **云端般的数据安全**：在书库顶部工具栏的右侧新增了“数据设置”齿轮按钮，点击即可呼出下拉菜单。
* **备份与恢复**：支持一键将您的书架书籍、阅读进度、所有书签及全局设置导出为备份文件；在换机或重新安装时，能够一键恢复，并能智能配对本地物理文件，重新解析出完整的图书封面及作者等元数据。
* **一键还原初始状态**：提供彻底清空书籍及进度的还原功能，并带有贴心的安全确认和“备份并还原”选项，避免误操作。

## 🛠️ 项目技术栈

* **桌面框架**：[Electron](https://www.electronjs.org/) (保障应用在桌面端的稳定表现)
* **前端界面**：[React](https://react.dev/) + [Vite](https://vite.dev/) (极速的开发与热更新体验)
* **状态管理**：[Zustand](https://github.com/pmndrs/zustand)
* **格式解析**：
  * **EPUB**: [Epub.js](http://epubjs.org/)
  * **PDF**: [PDF.js](https://mozilla.github.io/pdf.js/)
  * **MOBI / AZW3**: 自主开发的原生 Records 结构及 Huffman 解密提取引擎。
  * **TXT**: 支持多编码自动探测加载（避免乱码），并支持章节智能物理分片渲染。

## 🚀 快速开始

### 1. 检出与环境准备
确保本地安装了 [Node.js](https://nodejs.org/) (推荐 v18+)。

```bash
# 进入项目根目录
cd reader

# 安装依赖
npm install
```

### 2. 启动开发模式
```bash
npm run dev
```

### 3. 生产编译与打包
```bash
# 仅执行打包编译
npm run build

# 执行构建并打包为 Windows 独立运行程序 (.exe)
npm run package
```

### 4. 自动化发布 (仅开发人员)
```bash
# 执行打包并将构建出的最新版 zip/exe 上传至 GitHub Release
npm run publish
```
