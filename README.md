# YouQian Reader 📚 (游潜阅读器)

[![Version](https://img.shields.io/badge/version-1.4.1-blue.svg)](package.json)
[![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)](package.json)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](package.json)

**YouQian Reader** 是一款专为极致阅读体验量身打造的桌面电子书阅读器。它完美支持 **EPUB, MOBI, AZW3, PDF, TXT** 等主流电子书格式，配备了豪华的学术级纸质排版系统和毫秒级响应的翻页交互，在保持 60Hz 帧率下带来实体书般的纯净与舒适。

---

## 🌟 核心特色 (V1.4.1 性能版)

### 1. 物理滑页与自适应底栏章节追踪
- **CSS Column 多栏横向分页**：采用优雅的多栏流式布局（Columns），让文字自然分布在左右视口，带来横向翻页的实体书美感。
- **自适应窗口拉伸**：支持键盘、滚轮及绝对定位导航键，完美贴合窗口尺寸变化，横向对齐无错位。
- **高精度自适应章节名追踪**：针对 MOBI/AZW3 格式原生 `filepos` 字节偏移特性，引入**“文本脱水倒序模糊匹配定位”**，在加载时预先构建首行章节 DOM 引用映射缓存。无论是点击目录跳转还是滑动翻页，底部状态栏章节名均能实时 100% 零偏差动态同步显示（彻底告别一直卡在“正文”的现象）。

### 2. 学术著作级物理注释重构 (鼠标跟随与防遮挡)
- **纯文字上标替换与气泡移动跟随**：扫除垃圾注脚图片，重写为高精细纯文字上标。鼠标悬浮时气泡 0.1 毫秒瞬时呈现，并跟随鼠标移动，在样式底层移除了 `transform` 延迟，消除粘滞感。
- **屏幕位置感知自适应朝向**：智能判断鼠标物理位置，若鼠标偏左则气泡自动展示在光标右侧，若鼠标偏右则气泡自动展示在光标左侧，提供极致顺畅的防遮挡阅读视野。
- **局部按序重排**：在各章节内部重排注释序号，自动按 `注¹`、`注²` 重置，与高档实体纸质书编排规范 100% 契合。
- **本章注释跨栏面板**：在每一章正文结束、下一章标题开始前的缝隙中，智能植入一个磨砂玻璃般的【本章注释】面板。支持 `column-span: all` 物理跨栏排版，彻底解决注释段落在翻页时被拦腰竖向切断的排版冲突，同时物理隐藏全书最末尾长达几十页的垃圾冗余，使全书精简清爽。

### 3. 主进程字节级高精度物理锚点植入
- **破案底层偏置机制**：MOBI/AZW3 专属 `filepos` 跳转超链接（例如 `filepos=56486`）是指向原始合并记录字节流中的**字节偏移量（Byte Offset）**。由于中文在多字节编码（如 UTF-8）下占 3 字节，解密解码成 JS 字符串后会产生数万个位置的字符索引偏置，造成前端锚点插入位置完全错位，从而误触了目录页上的超链接自身（表现为点击目录“原地踏步、跳回目录页自身”的严重 Bug）。
- **字节级倒序分割注入**：我们在主进程 [mobi.js](electron/main/parsers/mobi.js) 的解密内容合并解压后，在字节 `Buffer` 层面，按从大到小（降序）的顺序对原始字节流进行 `slice` 物理切割，在精确的字节偏移处拼入 `<span id="filepos-VAL">` 定位锚点字节，再统一解码。
- **0 偏差直连跳转**：生成的 HTML 在对应章节的绝对物理起始位置天生且毫无偏置地嵌套着唯一的物理锚点。前端 `MobiReader` 彻底释放运算负担，跳转时直接通过 `document.getElementById('filepos-VAL')` 极速直连检索，**100% 实现数学级的零误触、零跳转偏移**。

### 4. 极致 O(1) 缓存与异步防抖联动 (V1.4.1 性能狂飙 ⚡)
- **O(1) 节点引用缓存**：以前在每次翻页时，章节目录联动（计算当前章节名）都会高频同步执行 DOM 树查询，对于拥有几百个章节、数万个 DOM 节点的大书极易导致严重的卡顿。现在在首次定位查找到章节目标元素后，**立即将其存入组件 `headingMapRef` 缓存**。后续翻页切换时直接以 O(1) 复杂度检索，耗时从几百毫秒骤降至 **0.01 毫秒**。
- **异步非阻塞防抖 (Debounced UI)**：将繁重的章节目录高亮与名称计算放入 100ms 的异步防抖延时中执行。翻页时主线程优先在 5ms 内完成页面切换渲染，待浏览器空闲后再静默高亮目录，从底层确保了翻页的**瞬间响应**。

### 5. 智能巨著性能降级层 (V1.4.1 性能狂飙 ⚡)
- **智能跳过动效**：当检测到当前书籍的总页数较多（超过 350 页，例如套装合集书《Python编程三剑客》）或内容极其庞大时，`triggerPageTransition` 会自动进入“极速秒翻模式”，完全停用外层卡片淡入淡出的重度不透明度重绘。
- **禁用过渡重绘**：在多栏流式分页布局下，超大书籍会自动将平移过渡样式（`transition`）设为 `none`，使页面切换直接在 GPU 纹理层瞬间完成，消灭重排掉帧，保证即使在几百万字的大书上也拥有 **60FPS 物理满帧** 的脆爽体验。

### 6. 异步分步页码测算与加载
- **异步页码测算提示**：在开书或动态调整字号、行高等排版参数时，状态栏在排版未完全测算就绪前智能显示“*页码测算中...*”，并在就绪后瞬时切换，完全避免了旧版在排版计算期间按钮禁用失效或数据错位的问题。
- **状态驱动翻页**：翻页动作统一升级为抽象化的状态驱动逻辑（'home', 'prev', 'next', 'end'），消灭同步调用中可能出现的闪烁或黑屏现象。

## 🛠️ 项目技术栈

- **桌面外壳**：[Electron](https://www.electronjs.org/) (渲染进程多线程隔离)
- **前端视图**：[React](https://react.dev/) + [Vite](https://vite.dev/) (毫秒级热更新开发)
- **全局状态**：[Zustand](https://github.com/pmndrs/zustand)
- **格式解析**：
  - EPUB: [Epub.js](http://epubjs.org/)
  - PDF: [PDF.js](https://mozilla.github.io/pdf.js/)
  - MOBI / AZW3: 自主实现的原生记录（Records）及 Huffman/CDIC 精密解密提取引擎
  - TXT: 多编码自动探测加载机制 (iconv-lite + chardet)，且支持章节物理分片渲染
- **自动化工具**：
  - 配备 [publish.js](scripts/publish.js) 自动化打包发布工具。自动利用 Git 代理（`SOCKS5`），使用原生 `curl.exe` 精准上传当前版本的 `.exe` 和 `.zip` 绿色版资产至 GitHub Release，自带进度条反馈。

## 🚀 快速开始

### 1. 检出与环境准备
确保本地安装了 [Node.js](https://nodejs.org/) (推荐 v18+)。

```bash
# 进入项目根目录
cd youqian-reader

# 安装依赖
npm install
```

### 2. 启动开发模式
```bash
npm run dev
```

### 3. 生产编译与打包
```bash
# 仅执行打包编译
npm run build

# 执行构建并打包为 Windows 独立运行程序 (.exe)
npm run package
```

### 4. 自动化打包发布 (对开发人员)
```bash
# 执行打包并将构建出的最新版 zip/exe 上传至 GitHub Release
npm run publish
```
